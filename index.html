Reproducible Metrics (Sample Code)


import math

def npv(rate, cashflows):
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows))

def irr(cashflows, guess=0.1):
    # Newton-Raphson
    r = guess
    for _ in range(100):
        f = sum(cf / (1 + r) ** t for t, cf in enumerate(cashflows))
        df = sum(-t * cf / (1 + r) ** (t + 1) for t, cf in enumerate(cashflows))
        if abs(df) < 1e-12:
            break
        r_next = r - f / df
        if abs(r_next - r) < 1e-10:
            return r_next
        r = r_next
    return r

cashflows = [-1000, 300, 350, 420, 520]
print("NPV @ 10%:", round(npv(0.10, cashflows), 2))
print("IRR:", round(irr(cashflows)*100, 2), "%")

Key Financial Formulae

Net Present Value (NPV): $$\text{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t}$$

Earnings Per Share (EPS): $$\text{EPS} = \frac{\text{Net Income} - \text{Dividends on Pref.}}{\text{Weighted Avg. Shares}}$$

Compound Annual Growth Rate (CAGR): $$\text{CAGR} = \left(\frac{V_{\text{final}}}{V_{\text{initial}}}\right)^{\tfrac{1}{n}} - 1$$
