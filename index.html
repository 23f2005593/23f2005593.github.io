CAGR Calculation Model

We use a standardized Python model for our growth calculations.


						def calculate_cagr(beginning_value, ending_value, num_years):
						  """
						  Calculates the Compound Annual Growth Rate (CAGR).
						  """
						  if beginning_value == 0 or num_years <= 0:
							return 0
						  
						  cagr = (ending_value / beginning_value) ** (1 / num_years) - 1
						  return f"{cagr:.2%}"

						# Example: Growth from $100M to $150M in 3 years
						print(calculate_cagr(100, 150, 3))
						# Expected Output: 14.47%
To ensure consistency in our reporting, all growth rates are calculated using this Python function. It provides a clear, repeatable method for determining compound annual growth.

Financial Formulas

Our projections are based on standard financial models, including the Compound Annual Growth Rate (CAGR).

The formula for CAGR is:

$$ CAGR = \left( \frac{EV}{BV} \right)^{\frac{1}{n}} - 1 $$

Where:

  • $EV$ = Ending Value
  • $BV$ = Beginning Value
  • $n$ = Number of Years
This is the mathematical representation of the code we just saw. It's a fundamental formula for measuring an investment's annual growth rate over a specified time period.
